## <p align="right">📋 حول المشروع</p>
<p align="right">
<b>مدربي دز</b> هي منصة إلكترونية جزائرية تهدف إلى جمع وتوثيق الكفاءات الرياضية في الجزائر. 
توفر المنصة مساحة للمدربين والرياضيين والمختصين في المجال الرياضي لعرض مهاراتهم وخبراتهم والتواصل مع بعضهم البعض.
</p>

## <p align="right">✨ المميزات</p>
<p align="right" dir="rtl">
- 🏅 إنشاء ملفات تعريفية للمدربين والرياضيين<br />
- 📊 عرض الخبرات والكفاءات الرياضية<br />
- 🔍 بحث وتصنيف حسب التخصص الرياضي<br />
- 📱 واجهة متجاوبة (Responsive)<br />
- 🌐 دعم اللغة العربية<br />
- 🔒 نظام تسجيل دخول آمن
</p>

## 🛠️ التقنيات المستخدمة

| التقنية | الاستخدام |
|---------|-----------|
| **Next.js 16** | إطار العمل الرئيسي |
| **TypeScript** | لغة البرمجة |
| **Tailwind CSS** | تصميم الواجهات |
| **Prisma 6** | إدارة قاعدة البيانات |
| **SQLite** | قاعدة البيانات |

## 🚀 كيفية التشغيل

```bash
# تثبيت الاعتماديات
npm install

# تشغيل وضع التطوير
npm run dev

# بناء المشروع للإنتاج
npm run build

# تشغيل النسخة الإنتاجية
npm start
```

افتتح [http://localhost:3000](http://localhost:3000) في المتصفح.
